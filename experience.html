MY JOURNEY

Experience in Web Development

I had an interesting experience while creating a portfolio website project for school. The project was part of my web development assignment, and I worked on it enthusiastically because I wanted to improve my coding skills. Before starting the project, I planned the design and thought about the content that I wanted to display on the website.

Before creating the website, I edited and designed the interface in Figma. I created a pink and white themed design to make the website look soft, aesthetic, and modern. I carefully arranged the layout, selected suitable fonts, and combined several colors to create a more attractive appearance.

I used HTML, CSS, and JavaScript to create the portfolio website. First, I made the homepage and added my profile information. After that, I designed the skills section, the landing page projects that I created during the second semester, and the contact page. I also added animations, color gradients, and modern layouts to make the website look more attractive and aesthetic.

During the coding process, I faced several problems. Sometimes the layout became messy, the animations did not work properly, and there were many errors in my code. I felt confused and tired, but I did not give up. I tried to fix the errors by checking the code carefully, watching tutorials, and asking my friends for help.

After spending many hours coding and editing the website, I finally completed the project successfully. I felt very proud and satisfied when I saw the final result of my portfolio website. The project looked modern, creative, and more attractive than I had imagined. From this experience, I learned many new things about web development, especially about patience, creativity, and problem-solving. This project also increased my interest in coding and motivated me to create even better websites in the future.
